All example .avs files expect "FLuaG.dll" in your Avisynth plugin folder for automatic plugin loading.
------------------------------------------------------------------------------------------------------
1_doc: example from documentation
2_info: stream and OpenGL informations
3_audio: introduction to audio editing
4_video: introduction to video editing
5_opengl: use of OpenGL for video editing
6_advanced: various effects, not for beginners anymore
7_engine: use of an engine (scenegraph)
8_karaoke: using FLuaG as karaoke effects maker
